# SalesPilot payment සහ activation වැඩ කරන විදිය

SalesPilot current subscription path එක Sri Lankan businesses bank transfers prefer කරන නිසා ඒකට design කරලා තියෙනවා. Plan එක තෝරලා amount එක transfer කරලා screenshot upload කරනවා. Review වුණාට පස්සේ activation code එක ලැබෙනවා.

## Current primary payment path

දැනට primary flow එක bank transfer proof. Local businesses වලට මේක simple සහ review trail එක clear.

## 1. Plan එක තෝරන්න

Dashboard එකේ Usage & Plan වෙත ගිහින් Subscribe open කරන්න. Starter හෝ Pro තෝරලා monthly හෝ annual billing select කරන්න.

Current public prices:

- Starter monthly: Rs. 1,490
- Starter annual: Rs. 14,900
- Pro monthly: Rs. 3,490
- Pro annual: Rs. 34,900

## 2. Exact amount එක transfer කරන්න

Dashboard එකේ පෙන්වන bank account එකට amount එක transfer කරන්න. හැකි නම් registered WhatsApp number එක payment reference ලෙස දාන්න.

Reference එක හරියට දාද්දී review team එකට payment proof එක correct business account එකට match කරන්න පහසුයි.

## 3. Payment screenshot upload කරන්න

Bank receipt එකේ clear JPG හෝ PNG screenshot එක upload කරන්න. App එක image upload accept කරලා proof එක admin review සඳහා යවනවා.

Screenshot එක unclear නම් proof එක reject විය හැකියි. එවිට නැවත submit කරන්න වෙනවා.

## 4. Verification සඳහා wait කරන්න

Submit කළාට පස්සේ payment status එක pending වෙනවා. SalesPilot team එක amount, reference, screenshot, selected plan check කරනවා.

Approve වුණාම SalesPilot activation code එක generate කරලා registered email එකට යවනවා.

## 5. Plan එක activate කරන්න

SalesPilot එකේ Activate Code tab එක open කරලා activation code එක enter කරන්න. Successful activation පස්සේ plan එක update වෙනවා.

Monthly plans දින 30ක් activate වෙනවා. Annual plans දින 365ක් activate වෙනවා.
