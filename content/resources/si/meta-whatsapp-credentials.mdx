# Phone Number ID, WABA ID සහ System User Access Token ගන්න විදිය

SalesPilot official Meta WhatsApp Cloud API එකට connect වෙනවා. Connect කරන්න `phone_number_id`, `waba_id`, සහ `access_token` කියන values තුන ඕනේ.

## පටන් ගන්න කලින්

- Business එකට අයිති Meta Business Manager account එකක් use කරන්න.
- WhatsApp number එක WhatsApp Business Platform සඳහා ready කරලා තියෙන්න ඕනේ.
- Production සඳහා temporary test token එකක් නෙවෙයි, System User token එකක් use කරන්න.
- Token එක private තියාගන්න. Token එක තියෙන කෙනෙකුට WhatsApp API permissions හරහා action කරන්න පුළුවන්.

## 1. Meta Business Manager open කරන්න

[Meta Business Manager](https://business.facebook.com/) වෙත ගිහින් ඔබේ WhatsApp account එක අයිති business portfolio එක select කරන්න.

Business portfolio එකක් නැත්නම් මුලින් create කරන්න. Verification සහ payment provider checks වලට use කරන legal business name එකම use කිරීම හොඳයි.

## 2. WABA ID එක හොයන්න

Business Settings open කරලා WhatsApp Accounts වෙත යන්න. ඔබේ business එකට connected WhatsApp Business Account එක select කරන්න.

WhatsApp Business Account ID එක තමයි WABA ID. මේක long numeric value එකක්. SalesPilot එකට copy කරද්දී හරියටම copy කරන්න.

## 3. Phone Number ID එක හොයන්න

WhatsApp Manager හෝ Meta app dashboard එක open කරලා WhatsApp API setup area එකට යන්න. Customers message කරන live phone number එක select කරන්න.

Phone Number ID කියන්නේ ඔබේ real phone number එක නෙවෙයි. Meta API එක use කරන long numeric ID එකක්. Temporary test number එකේ ID එක නෙවෙයි, live business number එකේ ID එක copy කරන්න.

## 4. System User එකක් create කරන්න

Business Settings තුළ Users, ඉන්පසු System Users open කරන්න. WhatsApp integration එක සඳහා System Admin user එකක් create කරන්න.

ඒ user ට WhatsApp account සහ Cloud API app එකට අවශ්‍ය asset access assign කරන්න.

## 5. Access token එක generate කරන්න

System User සඳහා new token එකක් generate කරන්න. මේ permissions දෙන්න:

- `whatsapp_business_messaging`
- `whatsapp_business_management`
- `business_management` Meta business asset access ඉල්ලුවොත්

Token එක copy කරලා safe place එකක තියාගන්න. SalesPilot save කරන්න කලින් Meta එක්ක token එක verify කරනවා.

## 6. SalesPilot එකට connect කරන්න

SalesPilot dashboard එකේ WhatsApp Connection වෙත යන්න. මේවා paste කරන්න:

- Phone Number ID
- WABA ID
- System User Access Token

Verify & Connect click කරන්න. SalesPilot credentials Meta එක්ක test කරලා WABA එක SalesPilot webhook එකට automatically subscribe කරනවා.

## සාමාන්‍ය වැරදි

- Test API setup page එකේ temporary token එක use කිරීම.
- Phone Number ID වෙනුවට display phone number එක copy කිරීම.
- Live number ID වෙනුවට test number ID copy කිරීම.
- WhatsApp messaging සහ management permissions නැති token එකක් create කිරීම.
- SalesPilot connect කළාට පස්සේ system user token එක revoke කිරීම.
