# How to get your Phone Number ID, WABA ID, and System User Access Token

SalesPilot connects to the official Meta WhatsApp Cloud API. To connect your business, you need three values: `phone_number_id`, `waba_id`, and `access_token`.

## Before you start

- Use a Meta Business Manager account owned by the business.
- Make sure your WhatsApp number is ready for the WhatsApp Business Platform.
- Use a System User token for production, not a temporary test token.
- Keep the token private. Anyone with the token can act through your WhatsApp API permissions.

## 1. Open Meta Business Manager

Go to [Meta Business Manager](https://business.facebook.com/) and select the business portfolio that owns your WhatsApp account.

If you do not have a business portfolio yet, create one first. Use the legal business name you plan to use for verification and payment provider checks.

## 2. Find the WABA ID

Open Business Settings, then go to WhatsApp Accounts. Select the WhatsApp Business Account connected to your business.

The WhatsApp Business Account ID is the WABA ID. It is a long numeric value. Copy it exactly into SalesPilot.

## 3. Find the Phone Number ID

Open WhatsApp Manager or your Meta app dashboard and go to the WhatsApp API setup area. Select the phone number customers will message.

The Phone Number ID is not your real phone number. It is a long numeric ID used by Meta's API. Copy the ID for the live business number, not the temporary test number.

## 4. Create a System User

In Business Settings, open Users, then System Users. Create a System Admin user for your WhatsApp integration.

Assign the correct business assets to that user. The system user needs access to the WhatsApp account and app used for Cloud API messaging.

## 5. Generate the access token

Generate a new token for the System User. Grant these permissions:

- `whatsapp_business_messaging`
- `whatsapp_business_management`
- `business_management` when Meta asks for business asset access

Copy the token once and store it safely. SalesPilot verifies the token with Meta before saving the connection.

## 6. Connect in SalesPilot

Open your SalesPilot dashboard and go to WhatsApp Connection. Paste:

- Phone Number ID
- WABA ID
- System User Access Token

Click Verify & Connect. SalesPilot tests the credentials with Meta and subscribes the WABA to the SalesPilot webhook automatically.

## Official references

- [Meta WhatsApp Cloud API documentation](https://developers.facebook.com/docs/whatsapp/cloud-api/)
- [Meta Business Manager](https://business.facebook.com/)

## Common mistakes

- Using the temporary token from the test API setup page.
- Copying the display phone number instead of the Phone Number ID.
- Copying the test number ID instead of the live number ID.
- Creating a token without WhatsApp messaging and management permissions.
- Revoking the system user token after SalesPilot has been connected.
